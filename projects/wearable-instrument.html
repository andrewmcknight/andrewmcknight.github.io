Wearable Electronic Instrument

Overview

A glove-based electronic musical instrument that translates hand, arm, and finger movements into sound in real time. The instrument is designed to be expressive and playable — mapping physical gesture to musical parameters like pitch, volume, timbre, and effects through customizable gesture-to-sound profiles.

Technical Details

  • Flex sensors on each finger detect individual finger bend angles
  • IMU captures hand and arm orientation, acceleration, and gesture detection
  • Microcontroller performs sensor fusion and signal processing on-device
  • Sensor data maps to musical parameters — pitch, volume, timbre, effects — with customizable gesture-to-sound profiles
  • Companion web application with BLE connectivity for real-time parameter control, sound mapping configuration, and visual feedback

Status

In progress as of Spring 2026. Hardware integration and sensor fusion are underway; web app interface and BLE communication pipeline are in active development.
